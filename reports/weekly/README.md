This directory contains weekly bridge monitoring reports.

## Structure
- `YYYY-WW/` - Reports organized by year and week number
- `latest/` - Symlink to the most recent weekly report

## File Naming Convention
- `STW-W-YYYYWW-BRIDGEID.md` - Markdown format
- `STW-W-YYYYWW-BRIDGEID.txt` - Plain text format

## Generation Time
Weekly reports are generated every Monday at 00:00 UTC.

## Retention
Weekly reports are kept for 1 year, then moved to archives.
