This directory contains daily bridge monitoring reports.

## Structure
- `YYYY-MM-DD/` - Reports organized by date
- `latest/` - Symlink to the most recent daily report

## File Naming Convention
- `STW-D-YYYYMMDD-BRIDGEID.md` - Markdown format
- `STW-D-YYYYMMDD-BRIDGEID.txt` - Plain text format

## Generation Time
Daily reports are generated every day at 00:00 UTC.

## Retention
Daily reports are kept for 90 days, then moved to archives.
